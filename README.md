# Requirements

- installed go
- installed git
- basic familiarity with git and github
- stable internet connection

# Workflow

- make changes in source code ([main.go file](./main.go))

- run tests locally (optional) using `.pathly/test.sh` command

- submit exercise using `.pathly/submit.sh` command

**Note**: Above steps are for linux users, if you are on windows run scripts using `.\<script-name>.bat` command where `<script-name>` is a script filename (example: `.pathly\test.bat`).
